# Sourcemaps Archive Team 🗺️

We are a dedicated collective focused on the preservation and archiving of frontend source maps. Our mission is to bridge the gap between minified production code and its original, human-readable source.

### 🔍 What We Do
* **Private Archiving:** Securely storing source maps for internal research and historical records.
* **Public Archiving:** Providing unminified versions of frontend code for educational and transparency purposes (where permitted).
* **Tooling:** Developing scripts to efficiently fetch and map `sourceMappingURL` references.

### 🛠️ How to Contribute
If you have found a source map that needs archiving, please open an issue in our `archive-requests` repository.

### 🛠️ How to Contribute Privately
